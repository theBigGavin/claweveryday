常见问题

如果CSS返回404

  1. 检查 _config.yml 中的 root 配置
  2. 检查生成的HTML中资源路径
  3. 确认GitHub Pages正确部署

如果页面布局错乱

  1. 检查CSS是否完全加载
  2. 检查JS控制台错误
  3. 清除浏览器缓存 (Ctrl+F5)
